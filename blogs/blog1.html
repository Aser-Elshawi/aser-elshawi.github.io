Printing Electric Jet Engine

Author: Aser Elshawi

last updated on: 5/17/2020

Is it possible to 3D print electric engine from scratch? While searching I came accross the following interesting page Amazing DIY Projects so I decided to download the stl (3D) files and try it with the help of my friend.

Main build components:

  • 3D plastic prints
  • Magnetic disks
  • Drill rod
  • Enameled magnetic wire

In order to get the 3D parts printed, I'm using my Ender 3 Pro printer:

A simple DC motor consists mainly of two parts:

  • Permanent magnet Stator
  • Tow pole electrically wired Rotor

Fairly simple design, the only tricky part here is delivering the current to wires on the rotor to keep the magnetic field and thus keep the momentum. That’s when the brush comes into the picture, with springs pushing the brush into the rotor the electric current is delivered, but what if we can do without the brush?

Yes, simple this could be achieved if we move the permanent magnet to the rotor, and connect the wires directly to the stator instead. Great! But is this possible?

Well the only problem would be controlling the current through the coils on the stator to change the direction during the rotor rotation. Luckily we can achieve this with an Arduino and ESC (Electric Speed Controller).

The Arduino would simply provide a PWM signal to the ESC, and the ESC will take care of the rest.

The Arduino code: 

#include <Servo.h>

Servo myservo;  // create servo object to control a servo
     
int potpin = 0;  // analog pin used to connect the potentiometer
int val;    // variable to read the value from the analog pin

void setup() {
  myservo.attach(9);  // attaches the servo on pin 9 to the servo object
}

void loop() {
  val = analogRead(potpin);            // reads the value of the potentiometer (value between 0 and 1023)
  val = map(val, 0, 1023, 0, 180);     // scale it to use it with the servo (value between 0 and 180)
  myservo.write(val);                  // sets the servo position according to the scaled value
  delay(15);                           // waits for the servo to get there
}
